This is An Embodied Conversational Agent just for motiviating users in physical fitness. The agent is bilingual(Converses in English and Bengali) and it takes date from custom datasets. I have applied RAG Architecture here, using Chroma Db as vector Database and Langchain for orchestrating. I also used FASTAPI for the backend, Using grokAPI for smoother and faster responses, i have llama3.2(8B) as the large language model.
